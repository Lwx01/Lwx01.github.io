什么是Hexo

​ Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

​ 本篇文章参考文档 | Hexo编写。

安装Node.js

​ Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)

​ Node.js 为大多数平台提供了官方的 安装程序。对于中国大陆地区用户，可以前往 淘宝 Node.js 镜像 下载。
